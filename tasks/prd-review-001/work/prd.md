# PRD：会员积分兑换功能（v0.3 草稿）

## 背景

会员活跃度下滑，运营希望上线积分兑换：会员用积分兑换优惠券或实物礼品，提升粘性。

## 功能描述

1. 会员在「积分商城」页浏览可兑换商品（优惠券 / 实物礼品）。
2. 每个商品标注所需积分与库存数量。
3. 会员点击「立即兑换」，系统扣减积分并扣减库存，生成兑换记录。
4. 优惠券即时到账；实物礼品由运营线下发货，会员可在「我的兑换」查看记录。
5. 兑换记录字段：记录 ID、商品名称、消耗积分、兑换时间。

## 积分获取（已有能力，仅引用）

- 签到 +5 分/日；下单按实付金额 1 元 = 1 分。

## 页面原型

- 积分商城列表页、商品详情页、兑换成功页（原型图另附）。
