# Release Notes v3.0

## 特性

- 引入异步导出中心，支持大数据量报表下载。
- 新增只读副本路由，读接口默认走副本。

## 变更

- **运行时基线升级：Java 最低版本从 11 提升到 17**，旧 JRE 启动将直接失败。
- **默认监听端口从 `8080` 改为 `9090`**：容器编排与反向代理需要同步调整；
  可通过 `server.port` 显式覆盖。
- 移除 v2 保留的 `logging.legacy` 开关，日志仅支持 JSON 格式。
- 缓存实现由本地 Caffeine 换成 Redis（对外行为不变，无需调用方调整）。

## 修复

- 修复导出任务偶发重复执行的问题。
